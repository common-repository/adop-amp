== Description ==
Get more revenue from your AMP site!

This plug in for site owners who haven’t converted into AMP site yet.
ADOP builds AMP site from your original page and helps to optimize your ads revenue derived from site. ADOP is a professional SSP (Supply Side Platform) in ad tech industry and ready to maximize your profit.


Needs to know :
If you already have built AMP site by yourself, this plug-in would not be needed. But there is a product, ‘ADOP AMP Ads’ that optimizes your site revenue on your AMP page. 
If you are interested in AMP ads, please talk to us!

And if you have any question, please don’t hesitate to post on Support or contact to our sales team. (sales@adop.cc)

== Installation ==
How to start :
After consulting with our support team, you will get new AMP domain link.
Install and activate the plugin.
On plug-in setting, please select template layout and fill the AMP domain link issued by ADOP team.
Please fill the blanks to configure plugin setting.
AMP Header Tag Domain : Please insert new AMP domain (it will be issued by ADOP Sales team)
Start Date : Please insert the starting date when ADOP AMP plugin is applied.

= What is AMP? =
* AMP is shorten word for ‘Accelerated Mobile Page’ which is differentiated from existing web pages.
AMP helps SEO and gets more traffic, since its page is speedy.
AMP is built with optimized combination of web components. In this way, a page can be rendered quickly, which is one of features Google prefers.

= What is ADOP AMP Plug in? =
* ADOP builds AMP site for you and this plugin is needed to be input on your site.
